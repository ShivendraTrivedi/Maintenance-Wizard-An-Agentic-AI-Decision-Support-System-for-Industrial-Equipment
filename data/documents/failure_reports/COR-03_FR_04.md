# Failure Analysis Report — COR-03 — FR-20240113-004

**Asset:** COR-03 (Coke Oven Pushing Ram)  
**Area:** Coke Making  
**Date of failure:** 2024-01-13  
**Downtime:** 36 hours  
**Fault mode:** Drive gearbox wear  

## Event Description
Operations reported an abnormal condition on COR-03. Rising current and vibration signal gear tooth wear. The parameter 'ram_force' deviated from its normal band (100–350 kN) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **lubrication failure**, leading to drive gearbox wear.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Rail clamp).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'ram_force' to the early-warning watch list with alert at 332 kN.
- Ensure spare 'Rail clamp' is held in stock given lead time.
