# Failure Analysis Report — CWP-07 — FR-20240208-002

**Asset:** CWP-07 (Cooling Water Pump)  
**Area:** Utilities  
**Date of failure:** 2024-02-08  
**Downtime:** 18 hours  
**Fault mode:** Impeller wear  

## Event Description
Operations reported an abnormal condition on CWP-07. Reduced flow and pressure from worn impeller clearances. The parameter 'discharge_pressure' deviated from its normal band (4–9 bar) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **long service**, leading to impeller wear.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Mechanical seal).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'discharge_pressure' to the early-warning watch list with alert at 8 bar.
- Ensure spare 'Mechanical seal' is held in stock given lead time.
