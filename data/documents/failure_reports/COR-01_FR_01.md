# Failure Analysis Report — COR-01 — FR-20250911-001

**Asset:** COR-01 (Coke Oven Pushing Ram)  
**Area:** Coke Making  
**Date of failure:** 2025-09-11  
**Downtime:** 2 hours  
**Fault mode:** Hard pushing / sticky oven  

## Event Description
Operations reported an abnormal condition on COR-01. Abnormally high ram force indicates poor coke release. The parameter 'ram_speed' deviated from its normal band (8–18 m/min) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **wall carbon build-up**, leading to hard pushing / sticky oven.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Rail clamp).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'ram_speed' to the early-warning watch list with alert at 17 m/min.
- Ensure spare 'Ram shoe' is held in stock given lead time.
