# Failure Analysis Report — SP-IDF-03 — FR-20250216-004

**Asset:** SP-IDF-03 (Sinter Plant ID Fan)  
**Area:** Sinter Making  
**Date of failure:** 2025-02-16  
**Downtime:** 24 hours  
**Fault mode:** Impeller erosion  

## Event Description
Operations reported an abnormal condition on SP-IDF-03. Abrasive dust thins blades, shifting balance over time. The parameter 'bearing_temperature' deviated from its normal band (40–80 degC) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **long service hours**, leading to impeller erosion.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Fan impeller).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'bearing_temperature' to the early-warning watch list with alert at 76 degC.
- Ensure spare 'Plummer block bearing' is held in stock given lead time.
