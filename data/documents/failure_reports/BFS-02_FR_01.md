# Failure Analysis Report — BFS-02 — FR-20250604-001

**Asset:** BFS-02 (Blast Furnace Stove)  
**Area:** Iron Making  
**Date of failure:** 2025-06-04  
**Downtime:** 18 hours  
**Fault mode:** Shell hot-spot  

## Event Description
Operations reported an abnormal condition on BFS-02. Localised refractory loss raises shell temperature with risk of breakthrough. The parameter 'shell_temperature' deviated from its normal band (60–120 degC) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **refractory wear**, leading to shell hot-spot.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Checker bricks (silica)).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'shell_temperature' to the early-warning watch list with alert at 114 degC.
- Ensure spare 'Hot blast valve seat' is held in stock given lead time.
