# Failure Analysis Report — BFS-03 — FR-20250206-004

**Asset:** BFS-03 (Blast Furnace Stove)  
**Area:** Iron Making  
**Date of failure:** 2025-02-06  
**Downtime:** 36 hours  
**Fault mode:** Hot blast valve leakage  

## Event Description
Operations reported an abnormal condition on BFS-03. Worn valve seat allows hot blast to bypass, lowering blast temperature. The parameter 'shell_temperature' deviated from its normal band (60–120 degC) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **refractory spalling**, leading to hot blast valve leakage.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Checker bricks (silica)).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'shell_temperature' to the early-warning watch list with alert at 114 degC.
- Ensure spare 'Cooling stave' is held in stock given lead time.
