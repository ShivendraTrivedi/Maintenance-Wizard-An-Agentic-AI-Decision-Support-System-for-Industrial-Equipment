# Failure Analysis Report — CWP-05 — FR-20250807-002

**Asset:** CWP-05 (Cooling Water Pump)  
**Area:** Utilities  
**Date of failure:** 2025-08-07  
**Downtime:** 18 hours  
**Fault mode:** Impeller wear  

## Event Description
Operations reported an abnormal condition on CWP-05. Reduced flow and pressure from worn impeller clearances. The parameter 'vibration' deviated from its normal band (0.8–4.5 mm/s) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **erosion**, leading to impeller wear.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Pump impeller).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'vibration' to the early-warning watch list with alert at 4 mm/s.
- Ensure spare 'Mechanical seal' is held in stock given lead time.
