# Failure Analysis Report — SP-IDF-03 — FR-20250910-005

**Asset:** SP-IDF-03 (Sinter Plant ID Fan)  
**Area:** Sinter Making  
**Date of failure:** 2025-09-10  
**Downtime:** 2 hours  
**Fault mode:** Impeller dust build-up / imbalance  

## Event Description
Operations reported an abnormal condition on SP-IDF-03. Dust accretion on impeller raises vibration and unbalance. The parameter 'vibration' deviated from its normal band (1.0–6.0 mm/s) prior to the trip.

## Root Cause
Investigation concluded the primary root cause was **inadequate gas cleaning**, leading to impeller dust build-up / imbalance.

## Corrective Action Taken
- Isolated the asset per LOTO SOP.
- Replaced affected component (Plummer block bearing).
- Restored normal operating parameters and monitored for 24 hours.

## Preventive Recommendation
- Tighten inspection interval for the affected sub-assembly.
- Add 'vibration' to the early-warning watch list with alert at 5 mm/s.
- Ensure spare 'Fan impeller' is held in stock given lead time.
